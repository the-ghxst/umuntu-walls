# Dossier Images pour Umuntu Walls

## Instructions pour ajouter des images

Ce dossier contient toutes les images des villes africaines pour la galerie.

### Format des noms de fichiers

Les noms de fichiers doivent suivre ce format :
```
{ville}-{collection}.jpg
```

### Exemples :
- `abidjan-heritage.jpg`
- `stonetown-aquarelle.jpg`
- `dakar-heritage.jpg`
- `marrakech-aquarelle.jpg`
- `lagos-heritage.jpg`
- `accra-aquarelle.jpg`

### Spécifications des images

- **Format** : JPG, PNG ou WebP
- **Taille recommandée** : 800x600px minimum
- **Ratio** : 4:3 ou 16:9
- **Poids** : Max 500KB par image (optimisé pour le web)

### Comment ajouter une image

1. Depuis GitHub : Clique sur "Add file" > "Upload files"
2. Glisse-dépose tes images ici
3. Commit les changements
4. Vercel redéploiera automatiquement le site
